# Dossier : Sidebar de gestion des conversations

Objectif : permettre à l’utilisateur de gérer ses conversations via une sidebar de type Notion.

Fonctionnalités prévues :
- GET : Récupérer les dossiers de l’utilisateur
- POST : Créer un dossier
- PATCH : Renommer un dossier
- DELETE : Supprimer un dossier
- Affichage UI : liste des dossiers, bouton +, menu contextuel
